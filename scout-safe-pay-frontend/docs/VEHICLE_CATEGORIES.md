## API Endpoints
```
GET /api/vehicle-data/categories          - List all categories
GET /api/vehicle-data/makes/{category}    - Get makes for category
GET /api/vehicle-data/models/{cat}/{make} - Get models for make
GET /api/vehicle-data/category/{category} - Full category data
GET /api/vehicle-data/search/makes        - Search makes
```
